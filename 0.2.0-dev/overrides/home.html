The New Pathpy ¶

  • {% include ".icons/octicons/check-circle-16.svg" %}

    Easy to Use

    You can directly get started using PathpyG since it adapts the interfaces from commonly used frameworks. The graphs are based PyG and plotting works similar as in pandas.

  • {% include ".icons/simple/jupyter.svg" %}

    Great with Jupyter Notebooks

    This package is built with excellent support for Jupyter Notebooks. It offers interactive HTML visualizations that enhance data analysis and exploration via D3.js.

  • {% include ".icons/octicons/stopwatch-24.svg" %}

    GPU Accelerated

    Since PathpyG is based on Torch, you can run your computations on the GPU. Even computations that cannot run on the GPU are parallelized for optimal performance.

  • {% include ".icons/material/book-edit.svg" %}

    Optimal for Scientific Writing

    This package is ideal for scientific writing as it supports LaTeX as an export option. This allows for seamless integration of the network visualizations via PGF/TikZ in your documents.

  • {% include ".icons/material/graph.svg" %}

    Netzschleuder Support

    pathpyG offers easy access to over 250 real world datasets via Netzschleuder - a catalogue and repository of network datasets with the goal of aiding scientific research.

  • {% include ".icons/fontawesome/brands/git-alt.svg" %}

    Open Source

    Our package is open source, promoting collaborative development and transparency. This allows anyone to contribute, modify, and distribute the software, fostering innovation and rapid improvement.
